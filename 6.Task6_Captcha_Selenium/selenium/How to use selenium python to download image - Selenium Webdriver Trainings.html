How to use selenium python to download image

  • Madhuri Rachel
  • May 20, 2016
  • 0 comment

I need get captcha image from browser. I’ve got a uniform resource locator of this image, but this image transforms each upgraded period (uniform resource locator is continuous).

Is there any option to get an image from the browser (like ‘save image as’ switch)?

In the other palm, I believe it needs to be work:

get a screen shot of the browser
get location of image
Harvest captcha from screen capture utilizing OpenCV
Hyperlink of the powerful captcha – link

The issue was solved via screen capture:

browser.save_screenshot(‘screenshot.png’)
img = browser.find_element_by_xpath(‘//*[@id=”cryptogram”]’)
loc = img.location

image = cv.LoadImage(‘screenshot.png’, True)
out = cv.CreateImage((150,60), image.depth, 3)
cv.SetImageROI(image, (loc[‘x’],loc[‘y’],150,60))
cv.Resize(image, out)
cv.SaveImage(‘out.jpg’, out)

Here is an entire instance (utilizing yahoo’s recaptcha as a target):

import urllib
from selenium import webdriver

driver = webdriver.Firefox()
driver.get(‘http://www.google.com/recaptcha/demo/recaptcha’)

# get the image source
img = driver.find_element_by_xpath(‘//div[@id=”recaptcha_image”]/img’)
src = img.get_attribute(‘src’)

# download the image
urllib.urlretrieve(src, “captcha.png”)

driver.close()
